What is Wikimedia Design Codex?

Codex is the Design System for Wikimedia. It features a toolkit for building user interfaces, containing design tokens, Vue 3 components and a VitePress documentation site.

How do I install Codex in my project?

You can install it using npm or yarn:

npm install @wikimedia/codex @wikimedia/codex-design-tokens @wikimedia/codex-icons

Then import the components and styles in your application.
